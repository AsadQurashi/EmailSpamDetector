## Spam Email Detector (NLP)

This project is a console-based spam email classifier built using
Logistic Regression and Natural Language Processing techniques.

### Features
- Text preprocessing using CountVectorizer
- Logistic Regression model
- Spam probability & confidence levels
- Interactive email testing
- Top spam word analysis

### Tech Stack
- Python
- scikit-learn
- pandas
- NumPy

### Future Improvements
- TF-IDF Vectorizer
- Better evaluation metrics
- Web deployment (Streamlit/Flask)
